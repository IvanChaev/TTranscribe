================================================================================
                    ГОЛОСОВОЙ ТРАНСКРАЙБЕР (Whisper large-v3-turbo)
================================================================================

Программа для записи речи с микрофона или загрузки аудиофайлов и их быстрой
расшифровки в текст. Использует модель large-v3-turbo от OpenAI (через faster-whisper)
с ускорением на видеокартах NVIDIA (CUDA). Результат копируется в буфер обмена.

================================================================================
                         СИСТЕМНЫЕ ТРЕБОВАНИЯ
================================================================================
- ОС: Windows 10/11 (64-бит)
- Python 3.8 – 3.12 (рекомендуется 3.11)
- Микрофон (если планируете запись с микрофона)
- Для GPU-ускорения: видеокарта NVIDIA с поддержкой CUDA 11/12
- Для CPU-режима: достаточно процессора, но обработка будет медленнее

================================================================================
                         УСТАНОВКА И НАСТРОЙКА
================================================================================

1. Установите Python (если ещё не стоит)
   Скачайте с https://www.python.org/downloads/ и установите.
   ВАЖНО: при установке отметьте галочку "Add Python to PATH".

2. Установите FFmpeg (если ещё не стоит)
   Скачайте с https://ffmpeg.org/download.html (раздел Windows, сборки от gyan.dev или BtbN).
   Распакуйте архив в удобное место (например, C:\ffmpeg) и добавьте папку bin в системную переменную PATH.
   Как проверить: откройте командную строку и введите ffmpeg -version.

3. Установите зависимости Python
   Откройте командную строку (Win+R -> cmd) и выполните:
   pip install faster-whisper pyaudio pydub pyperclip tkinterdnd2

   Примечание: если pyaudio не устанавливается, скачайте готовый .whl файл
   для вашей версии Python с https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
   и установите его командой: pip install путь\к\файлу.whl

4. (Для владельцев NVIDIA) Настройка CUDA
   Чтобы расшифровка использовала видеокарту, необходимо установить CUDA Toolkit 12.x.
   - Скачайте с https://developer.nvidia.com/cuda-downloads (выберите Windows, exe local).
   - Установите, выбрав компоненты "CUDA" и "CUDA Tools" (остальное опционально).
   - После установки перезагрузите компьютер.
   
   Если CUDA Toolkit не устанавливается, но у вас уже есть библиотеки CUDA (например установлена в другом месте, то), 
   можно прописать путь к ним в системную переменную PATH или в bat-файл
   (см. раздел "Дополнительные настройки").

5. Настройка скрипта для вашего оборудования
   Откройте transcriber.py в блокноте и найдите строчки:
   model = WhisperModel("large-v3-turbo", device="cuda", compute_type="float16")
   Если у вас нет видеокарты NVIDIA, замените "cuda" на "cpu", а "float16" на "int8".
   Сохраните файл.

6. Первый запуск (скачивание модели)
   При первом запуске программа автоматически загрузит модель large-v3-turbo (~1.5 ГБ).
   Это происходит ОДИН раз и может занять несколько минут в зависимости от скорости интернета.
   Во время загрузки окно программы может временно не отвечать – это нормально.
   Если ничего не происходит лучше не прерывать процесс установки,
   смотрите так же в диспетчер задач при первых попытках загрузки/использования, а так же проверяйте стандартные путь установки.

================================================================================
                         ЗАПУСК ПРОГРАММЫ
================================================================================
- Поместите оба файла (transcriber.py и start_transcriber.bat) в одну папку.
- Запустите start_transcriber.bat двойным щелчком.
  Консольное окно быстро закроется, и откроется графический интерфейс.

================================================================================
                         ИСПОЛЬЗОВАНИЕ
================================================================================
• Запись с микрофона:
  Нажмите кнопку "🎤 Начать запись" – начнётся захват звука.
  Говорите в микрофон. Когда закончите, нажмите "⏹ Остановить запись".
  После короткой обработки расшифрованный текст окажется в буфере обмена(для проверки Win + V)
  (можно вставить Ctrl+V). Временный аудио-файл из которого происходит транскрибация удаляется автоматически.

• Загрузка аудиофайла:
  - Перетащите файл (wav, mp3, ogg, flac, m4a, aac) в область "Перетащите аудиофайл сюда".
  - Или нажмите "📁 Загрузить аудио" и выберите файл.
  После транскрибации текст будет скопирован в буфер обмена.

• Статус обработки отображается в нижней части окна.

================================================================================
                         ДОПОЛНИТЕЛЬНЫЕ НАСТРОЙКИ
================================================================================

1. Создание ярлыка с правами администратора (если нужно)
   Если вы хотите использовать nvidia-smi для ограничения энергопотребления видеокарты,
   создайте ярлык для start_transcriber.bat, откройте его свойства,
   вкладка "Ярлык" -> "Дополнительно" -> отметьте "Запуск от имени администратора".
   В сам bat-файл перед запуском добавьте, например:
   nvidia-smi -pl 125   (где 125 – лимит в ваттах для вашей видеокарты)
   Без этого программа работает в стандартном режиме без ограничений.

2. Пользовательские CUDA-библиотеки
   Если CUDA Toolkit не установлен, но библиотеки (cublas64_*.dll) находятся в особой папке,
   добавьте путь к этой папке в переменную PATH прямо в start_transcriber.bat:
   set "PATH=C:\путь\к\папке;%PATH%"

3. Смена языка распознавания
   По умолчанию установлен язык "ru". Для другого языка измените language="ru"
   в вызовах model.transcribe на нужный (например, "en").

================================================================================
                         ВОЗМОЖНЫЕ ПРОБЛЕМЫ
================================================================================
- "Pythonw.exe не найден" при запуске bat:
  Убедитесь, что Python добавлен в PATH. Если нет, укажите полный путь к pythonw.exe в bat.
- Ошибка "No module named 'faster_whisper'":
  Выполните pip install faster-whisper.
- Ошибка при записи: "No Default Input Device Available":
  Проверьте, подключён ли микрофон и разрешён ли доступ к нему в Windows.
- Ошибка "Library cublas64_*.dll is not found":
  Установите CUDA Toolkit (см. пункт 4 установки) или используйте CPU-режим.
- Долгая обработка:
  На CPU расшифровка может занимать в 5-10 раз больше времени, чем на GPU.
  Используйте более короткие аудиозаписи или приобретите совместимую видеокарту.
  Если что-то не получается, то вы можете все эти файлы(и эту инструкцию) закинуть в нейросеть(например общедоступный deepseek) для решения проблемы.
