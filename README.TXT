================================================================================
            ГОЛОСОВОЙ АССИСТЕНТ: ТРАНСКРИБАЦИЯ И TTS (Whisper large-v3-turbo)
Программа объединяет:
  1. Расшифровку речи (с микрофона или из аудиофайлов) в текст.
  2. Синтез речи (TTS) – озвучивание введённого текста голосом Дмитрия (Edge TTS).

По умолчанию всё работает на центральном процессоре (CPU) – видеокарта не требуется.
При желании можно ускорить расшифровку в 5–10 раз, подключив GPU NVIDIA (см. ниже).

Интерфейс в тёмной теме с красным текстом, drag-and-drop файлов, копирование в буфер.
================================================================================
                         СИСТЕМНЫЕ ТРЕБОВАНИЯ
- ОС: Windows 10/11 (64-бит)
- Python 3.8 – 3.12 (рекомендуется 3.11)
- Микрофон (для записи)
- Интернет-соединение (для TTS; для расшифровки не требуется после загрузки модели)
- Видеокарта НЕ ОБЯЗАТЕЛЬНА – работает на любом CPU
- Для ускорения: видеокарта NVIDIA с поддержкой CUDA 11/12
================================================================================
                         УСТАНОВКА И НАСТРОЙКА
1. Установите Python (если ещё нет)
   Скачайте с https://www.python.org/downloads/ и установите.
   ВАЖНО: отметьте "Add Python to PATH".

2. Установите FFmpeg (обязательно!)
   Скачайте с https://ffmpeg.org/download.html (Windows, сборка gyan.dev или BtbN).
   Распакуйте архив (например, в C:\ffmpeg) и добавьте папку bin в переменную PATH.
   Проверка: откройте командную строку и введите ffmpeg -version.

3. Установите зависимости Python
   Откройте командную строку и выполните:
   pip install faster-whisper pyaudio pyperclip tkinterdnd2 edge-tts pygame

   Если pyaudio не ставится, скачайте .whl под вашу версию Python с:
   https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
   и установите: pip install путь\к\файлу.whl

4. Первый запуск (автоматическая загрузка модели)
   При первом запуске программа скачает модель large-v3-turbo (~1.5 ГБ).
   Это делается один раз и требует интернета. Окно может временно не отвечать – ждите.
   Не прерывайте процесс! При проблемах проверьте диспетчер задач и папку кэша моделей.

5. (Опционально) Ускорение через GPU NVIDIA
   По умолчанию используется CPU. Если у вас видеокарта NVIDIA, можно ускорить расшифровку:
   - Установите CUDA Toolkit 12.x с https://developer.nvidia.com/cuda-downloads
     (выбрать Windows, exe local, компоненты "CUDA" и "CUDA Tools").
   - Перезагрузите компьютер.
   - Откройте transcriber_tts.py в блокноте и найдите строки:
        model = WhisperModel("large-v3-turbo", device="cpu", compute_type="int8")
     Замените "cpu" на "cuda", "int8" на "float16" (две строчки в функциях транскрибации).
   - Сохраните файл.
   После этого расшифровка будет использовать видеокарту и станет значительно быстрее.
================================================================================
                         ЗАПУСК ПРОГРАММЫ
- Поместите файлы transcriber_tts.py и start_ttstranscriber.bat в одну папку.
- Запустите start_ttstranscriber.bat двойным щелчком.
  Или запустите вручную: python transcriber_tts.py   (консоль будет видна).
================================================================================
                         ИСПОЛЬЗОВАНИЕ
Верхняя секция – синтез речи (TTS):
  • Введите текст в текстовое поле.
  • "📋 Вставить" – из буфера обмена.
  • "🔊 Озвучить" – генерирует и воспроизводит речь (голос Дмитрия, быстрая скорость).
  • "🗑 Очистить" – очищает поле.

Нижняя секция – транскрибация:
  • Запись с микрофона:
    Нажать "🎤 Начать запись" – говорить – нажать "⏹ Остановить запись".
    Расшифрованный текст автоматически копируется в буфер обмена (Ctrl+V).
  • Загрузка аудиофайла:
    - Перетащить файл в пунктирную область, или
    - Нажать "📁 Загрузить аудио".
    Поддерживаются форматы: wav, mp3, ogg, flac, m4a, aac.
    Текст копируется в буфер.

  • Статус отображается внизу окна.
================================================================================
                         ДОПОЛНИТЕЛЬНЫЕ НАСТРОЙКИ
1. Ограничение энергопотребления GPU (nvidia-smi)
   В start_ttstranscriber.bat перед строкой запуска можно добавить:
   nvidia-smi -pl 125   (125 – пример лимита в ваттах)

2. Пользовательские CUDA-библиотеки
   Если CUDA Toolkit не установлен, но библиотеки лежат в отдельной папке,
   добавьте путь к ней в PATH: set "PATH=C:\путь\к\cuda;%PATH%"

3. Изменение языка распознавания
   В transcriber_tts.py найдите language="ru" и замените на нужный код (например, "en").

4. Смена голоса или скорости TTS
   Найдите edge_tts.Communicate(...) в коде.
   Можно заменить "ru-RU-DmitryNeural" на другой голос (список: edge-tts --list-voices)
   и изменить rate="+50%" (например, "+0%" – обычная скорость).
================================================================================
                         ВОЗМОЖНЫЕ ПРОБЛЕМЫ
- "pythonw.exe не найден" – проверьте PATH, либо используйте python вместо pythonw.
- "No module named 'faster_whisper'" – выполните pip install faster-whisper.
- "No Default Input Device Available" – подключите микрофон или проверьте настройки.
- Ошибка библиотек cublas64_*.dll – либо установите CUDA Toolkit, либо используйте CPU.
- Долгая обработка на CPU – это нормально, используйте GPU для ускорения.
- TTS не работает без интернета – Edge TTS требует сеть.
- При любых сложностях обратитесь к нейросети (DeepSeek, ChatGPT) с этим readme.
